## Configured scripts

Before running any script install dependencies and configure [Husky Git hook](https://typicode.github.io/husky/#/):

```
npm i
npm run prepare
```

Local development mode:

```
npm run dev
```

Running tests:

```
npm run test
```

Get test coverage:

```
npm run coverage
```
